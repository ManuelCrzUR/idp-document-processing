# IDP Document Processing
## Manuel Cruz Garrote - Universidad del Rosario

Proyecto de Procesamiento Inteligente de Documentos enfocado en la detección e inpainting de sellos oficiales para mejorar el OCR.

### Estructura
- `data/`: Datasets (SROIE y Custom).
- `src/`: Código fuente modular.
- `notebooks/`: Experimentación.
- `tests/`: Pruebas unitarias.
- `docs/`: Documentación técnica.
